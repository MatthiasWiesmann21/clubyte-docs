# Theme Design

The **Theme Design** enables you to personalize the look and feel of your application by setting primary colors for buttons. This functionality ensures that your buttons adhere to your brand's design guidelines and are visually optimized for light and dark themes.

---

## Features

![Theme Design Settings Page](/Settings-ThemeDesign.png)

### Configure Primary Colors for Buttons
You can set distinct primary colors for buttons in:
- **Light Mode:** The color used when the application is displayed in a light theme.
- **Dark Mode:** The color used when the application is displayed in a dark theme.

These settings allow you to maintain a consistent and attractive design across different themes while improving user interaction.
